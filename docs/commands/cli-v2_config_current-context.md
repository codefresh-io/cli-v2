## cli-v2 config current-context

Shows the currently selected Codefresh authentication context

```
cli-v2 config current-context [flags]
```

### Examples

```

# Shows the current context:

        cli-v2 config current-context
```

### Options

```
  -h, --help   help for current-context
```
